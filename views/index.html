redirects.jit.su

Small, simple and free redirection service for the Node.js community.

Instructions

  1. Fork this repo: appsattic/redirects
  2. Edit redirects.js and add a key/value pair for the from/to domains (Note: do not edit package.json or _package.json)
  3. Commit and send a pull request
  4. Wait a day or so for me to re-release
  5. Set your DNS records on the domain you're redirecting from
    • naked domains: set the A records using dns.jit.su
    • subdomains: set the CNAME to redirects.jit.su
  6. Done

Note: I need pull requests since each deploy has to inform Nodejitsu which domains to listen on to direct to this application. Thanks.
